iceToast - 1.4

Basic usage
基础用法

iceToast.push( 'A massage' )

iceToast.push( 'long text ...' )

Basic usage, with params
基础用法，附带参数

iceToast.push( {words: 'A 10s massage', time: 10000 } )

iceToast.push( {words: 'A blue massage', bgcolor: 'blue' } )

iceToast.push( {words: 'A green massage', bgcolor: 'green' } )

iceToast.push( {words: 'A red massage', bgcolor: 'red' } )

iceToast.push( {words: 'A yellow massage', bgcolor: 'yellow' } )

Default & Available [ bgcolor ]: default, blue, green, red, yellow

Custom bgColor
自定义背景色

First, edit your iceToast Object
首先，修改 iceToast 对象
iceToast.baseConfig.colors.pink = '#FF8CB2';

Then use
然后执行
iceToast.push( {words: 'A pink massage', bgcolor: 'pink' } )

Global position
全局位置

iceToast.setPosition('top-right')

iceToast.setPosition('top-center')

iceToast.setPosition('bottom-right')

Available params: 'top-center', 'top-right', 'bottom-right'

You can custom 'iceToast.baseConfig' as your default values from the source code.
